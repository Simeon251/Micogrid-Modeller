# MicroGridModeler

A microgrid simulation and optimization project focused on modeling energy dispatch, battery behavior, and hybrid system performance.

## Key Modules
- `microgrid_simulation.py` - main simulation driver
- `dispatch_model.py` - dispatch optimization logic
- `hybrid_model.py` - hybrid system modeling
- `battery_module.py` - battery system behavior
- `Load_model.py` - load profile handling

## To Run
```bash
python microgrid_simulation.py
```
